D. Vacation

题意：有n+1n+1n+1辆车，每辆车有自己的长度，距离终点时间，速度。求第000辆车到达终点时间。其中不能超车，且车过终点后依旧向前行驶

思路：枚举每个车作为某个连续段的队头，然后取MaxMaxMax

E. Path

题意：破坏一条路径的代价为路径长度，求破坏所有最短路的最小代价

思路：跑一边最短路后重建图跑最大流

I. String

题意：构造一个子序列，长度为kkk，且各个字母iii，出现次数在li−ril_i-r_ili​−ri​之间

思路：贪心从小到大枚举每一位能否放在答案中，检查放入后剩余后缀串能否满足限制
